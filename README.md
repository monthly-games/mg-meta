# mg-meta

Monthly Games 비즈니스/프로세스/템플릿/규칙 정리용 저장소

## 구조

```
mg-meta/
  ├─ docs/
  │   ├─ business/           # 비전, 포트폴리오 전략
  │   ├─ process/            # 개발 프로세스, AI 방법론, 릴리즈 플레이북
  │   └─ templates/          # 이슈/GDD/마케팅 템플릿
  ├─ .github/
  │   └─ ISSUE_TEMPLATE/     # GitHub 이슈 템플릿
  └─ README.md
```
