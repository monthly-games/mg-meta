# MG-Games 화면 방향(Orientation) 가이드

> **문서 버전**: 1.0.0
> **최종 수정**: 2025-12-19
> **적용 대상**: 52종 전체 게임 (MG-0001 ~ MG-0052)

---

## 1. 개요

### 1.1 목적
각 게임의 장르, 조작 방식, 콘텐츠 특성에 따른 최적의 화면 방향(Portrait/Landscape)을 정의합니다.

### 1.2 화면 방향 분류

| 방향 | 설명 | 게임 수 |
|------|------|---------|
| **Portrait (세로)** | 9:16 ~ 9:21 비율, 한손 조작 | 46종 |
| **Landscape (가로)** | 16:9 ~ 21:9 비율, 양손 조작 | 4종 |
| **Flexible (유연)** | 양방향 지원, 적응형 UI | 2종 |

### 4.3 종횡비 대응

```dart
Widget buildAdaptiveLayout(BuildContext context) {
  final size = MediaQuery.of(context).size;
  final aspectRatio = size.height / size.width;  // Portrait 기준

  if (aspectRatio > 2.0) {
    // 세로로 긴 기기 (21:9 폴더블 등)
    return TallLayout();
  } else if (aspectRatio < 1.7) {
    // 세로로 짧은 기기 (태블릿 등)
    return WideLayout();
  } else {
    // 표준 비율 (16:9 ~ 19:9)
    return StandardLayout();
  }
}
```

class FoldableAwareWidget extends StatefulWidget {
  @override
  State<FoldableAwareWidget> createState() => _FoldableAwareWidgetState();
}

class _FoldableAwareWidgetState extends State<FoldableAwareWidget>
    with WidgetsBindingObserver {

  @override
  void didChangeMetrics() {
    // 화면 크기 변화 시 호출
    setState(() {});
  }

  @override
  Widget build(BuildContext context) {

    if (isUnfolded) {
      return UnfoldedLayout();  // 펼친 상태
    } else {
      return FoldedLayout();    // 접힌 상태
    }
  }
}

## 6. 회전 전환 처리

### 6.1 Flexible 게임 전환 애니메이션

```dart
class OrientationTransition extends StatelessWidget {
  final Widget portrait;
  final Widget landscape;

  @override
  Widget build(BuildContext context) {
    return OrientationBuilder(
      builder: (context, orientation) {
        return AnimatedSwitcher(
          duration: Duration(milliseconds: 300),
          transitionBuilder: (child, animation) {
            return FadeTransition(
              opacity: animation,
              child: child,
            );
          },
          child: orientation == Orientation.portrait
              ? portrait
              : landscape,
        );
      },
    );
  }
}
```

### 6.2 상태 보존

```dart
// 회전 시 게임 상태 보존
class GameState extends ChangeNotifier {
  // 회전에 영향받지 않는 데이터
  int score = 0;
  int level = 1;
  List<Item> inventory = [];

  // 회전 시 재계산 필요
  Offset? uiPosition;
  Size? buttonSize;

  void onOrientationChange(Orientation orientation) {
    // UI 위치만 재계산, 게임 데이터는 유지
    recalculateUIPositions(orientation);
    notifyListeners();
  }
}

## 7. 테스트 체크리스트

### 7.1 Portrait 게임 테스트

```
[ ] 세로 고정이 올바르게 적용되는가?
[ ] 세로로 긴 기기(21:9)에서 레이아웃이 정상인가?
[ ] 세로로 짧은 기기(태블릿)에서 레이아웃이 정상인가?
[ ] 한손 조작이 편리한가?
[ ] Safe Area가 올바르게 적용되는가?
```

### 7.2 Landscape 게임 테스트

```
[ ] 가로 고정이 올바르게 적용되는가?
[ ] 21:9 기기에서 양쪽 여백 처리가 정상인가?
[ ] 양손 조작 버튼 배치가 적절한가?
[ ] 컷아웃/노치 영역 처리가 정상인가?
[ ] 가상 버튼 간섭이 없는가?
```

```
[ ] Portrait ↔ Landscape 전환이 매끄러운가?

[ ] 전환 애니메이션이 300ms 이내인가?
[ ] 각 방향에서 UI가 최적화되어 있는가?
