## 1. Executive Summary

### 1.1 포트폴리오 구성

Monthly Games는 **36개 게임**으로 구성된 멀티 게임 포트폴리오로, 다음과 같이 분류됩니다:

| 구분 | 게임 수 | 특징 | 목표 |
|------|---------|------|------|
| **Year 1 (0001-0012)** | 12개 | 장르 다양성 실험 | 공통 모듈 검증, 시장 테스트 |
| **Year 2 (0013-0024)** | 12개 | 시스템 심화 | 공통 프레임워크 확장 |
| **Level A (0025-0036)** | 12개 | 수집형 JRPG | 글로벌 시장 공략, 고ARPU |

### 1.2 핵심 설계 철학

1. **공통 모듈 재사용**: 36개 게임이 `mg-common-game` 엔진을 공유
2. **장르 믹스**: 단일 장르가 아닌 2-3개 장르 융합 (예: Puzzle + Idle + JRPG)
3. **방치(Idle) 기반**: 대부분의 게임에 방치 요소 탑재로 세션 부담 최소화
4. **시너지 시스템**: 수집/편성/조합의 전략적 재미 강조
5. **Live Ops 구조**: 시즌/이벤트 중심의 지속적 업데이트 설계

## 3. Year 1 게임 상세 분석 (0001-0012)

### 3.1 MG-0001: 심플 타워 디펜스

#### 게임 루프
```
빌드 페이즈(준비) → 웨이브 방어(액션) → 골드 획득(보상) → 타워 업그레이드(성장) → 반복
```

#### 재미 요소 (Fun Pillars)
| 축 | 목표 | 핵심 경험 |
|----|------|----------|
| 전략적 배치 | 최적 타워 위치 발견 | "내 배치가 이 웨이브를 박살냈다!" |
| 경제 효율 | 골드 대비 데미지 최적화 | 비용 효율 극대화 |
| 위기 관리 | 누수 대응 | 긴급 스킬로 위기 극복 |

#### 아웃게임 요소
- **성장 시스템**: 타워 업그레이드, 신규 타워 해금
- **진행도**: 무한 웨이브 최고 기록
- **일일 루틴**: 일일 미션, 보상 수령

#### 레벨 디자인
- **웨이브 구조**: 점진적 난이도 상승 (수, 속도, HP)
- **맵 디자인**: 고정 경로 기반 그리드 배치
- **밸런스**: 20 Lives 시스템, 누수 1회당 -1 Life

---

### 3.2 MG-0002: 고양이 연금술 공방

#### 게임 루프
```
재료 수집(방치) → 레시피 실험/제작 → 결과물 판매 → 공방 업그레이드 → 반복
```

#### 재미 요소 (Fun Pillars)
| 축 | 목표 | 핵심 경험 |
|----|------|----------|
| 수집/조합 (Discovery) | 새 레시피 발견 | "이 조합은 뭐가 될까?" 기대감 |
| 성장/효율 (Growth) | 공방 효율화 | 처음엔 힘들었던 제작이 쉬워짐 |
| 표현/꾸미기 (Expression) | 개성 있는 공방 | 내 취향대로 꾸민 공방에 애착 |

#### 디자인 레버
- **힌트 기반 레시피 발견**: 완전 랜덤이 아닌 힌트 제공
- **대성공/실패 분산**: 150%/100%/50% 결과 분기
- **인테리어 → 생산 영향**: 풍수 보너스 시스템
- **고양이 특성**: 불꽃이(화염 특화), 눈송이(얼음 특화) 등

#### 아웃게임 요소
- **도감 시스템**: 레시피 도감, 3회 성공 시 자동 제작 해금
- **고양이 수집**: 고양이별 성격/전문 분야
- **공방 꾸미기**: 테마별 가구 세트

### 3.3 MG-0003: 픽셀 용병단 키우기

#### 게임 루프
```
방치 전투(자원 획득) → 용병단 강화 → 던전/보스 진행 → 기지 업그레이드 → 반복
```

#### 재미 요소 (Fun Pillars)
| 축 | 목표 | 핵심 경험 |
|----|------|----------|
| 성장/수집 (Growth) | 용병단 강화 체감 | 내 용병들이 강해져서 뿌듯 |
| 빌드/시너지 (Build) | 파티 구성 퍼즐화 | 최적 조합 발견 성취감 |
| 파훼/도전 (Challenge) | 강적 전략 돌파 | 빌드 변경으로 보스 클리어 |

#### 디자인 레버
- **역할 시너지**: 엘프 3인 → 치명타+10%, 기사단 → 방어+15%
- **방치 보상 리포트**: "용사 렉스가 드래곤 처치! MVP 선정"
- **관전 재미**: 30초 내 클리어 시 보너스
- **평행 성장축**: 본부 레벨 → 영웅 최대 레벨 상한

#### 아웃게임 요소
- **영웅 수집/승급**: R/SR/SSR 희귀도
- **장비 강화**: 세트 효과
- **스킬 트리**: 해금 시스템
- **본부/연구 트리**: 메타 성장

### 3.4 MG-0004: 카페 매치 타이쿤

#### 게임 루프
```
매치3 스테이지 → 카페 확장/인테리어 → 손님 응대/평점 관리 → 이벤트/단골 상호작용 → 반복
```

#### 재미 요소 (Fun Pillars)
| 축 | 목표 | 핵심 경험 |
|----|------|----------|
| 숙련/퍼즐 (Mastery) | 퍼즐 실력 향상 | 어려운 스테이지 클리어 성취 |
| 꾸미기 (Expression) | 개성 있는 카페 | 내 카페에 대한 애착 |
| 스토리/캐릭터 (Narrative) | 감정적 연결 | 단골 이야기가 궁금 |

#### 디자인 레버
- **카페 맥락 연결**: 커피콩 블록 → 커피 제조, 손님 주문 = 미션 목표
- **인테리어 세트 효과**: 빈티지 테마 → 문화인 방문률 UP
- **단골 손님 시스템**: 월요일 = 작가 지수씨 방문
- **경영 지표 연동**: 평점 4.5 이상 → VIP룸 해금

### 3.5 MG-0005: 로그라이크 퍼즐 던전

#### 게임 루프
```
턴제 퍼즐/전투 → 층별 보상 선택(스킬/유물) → 사망 시 메타 보상 → 메타 업그레이드 → 재도전
```

#### 재미 요소 (Fun Pillars)
| 축 | 목표 | 핵심 경험 |
|----|------|----------|
| 빌드/조합 (Build) | 매 런 다른 빌드 | "이번 런은 이 빌드로" |
| 도전/재도전 (Challenge) | 재도전 욕구 | "다음엔 더 잘할 수 있다" |
| 발견/학습 (Discovery) | 새 조합 발견 | "이런 조합이 되네!" |

#### 디자인 레버
- **테마 빌드 유도**: 첫 선택에서 방향 제시, 관련 옵션 확률 증가
- **사망 리포트**: "받은 독 데미지: 340 (전체 45%)"
- **후회 설계**: 선택 후 "그 유물이 있었으면..." 상황

#### 레벨 디자인
- **런 길이**: 15~25분
- **층 구조**: 점진적 난이도, 선택지 분기
- **메타 progression**: 10회 플레이 → 새 캐릭터 해금

### 3.6 MG-0006: 히어로 오토 배틀 아레나

#### 게임 루프
```
영웅 수집/강화 → 덱 편성/전략 설정 → PVP 매치/리그 참여 → 시즌 보상 → 반복
```

#### 재미 요소 (Fun Pillars)
| 축 | 목표 | 핵심 경험 |
|----|------|----------|
| 경쟁/랭킹 (Competition) | 랭킹 상승 긴장감 | 랭킹 올라갈 때 뿌듯 |
| 메타 읽기 (Meta) | 메타 분석/대응 | 카운터로 승리 쾌감 |
| 수집/강화 (Growth) | 영웅 수집 성장 | 새 영웅 획득 기대감 |

#### 디자인 레버
- **메타 순환 설계**: 가위바위보 상성, 버프/디버프 상쇄
- **덱 구성 제한**: 총 코스트 30, 같은 직업 최대 2명
- **리그 긴장감**: 3연승 → 승급전, 3연패 → 강등전
- **메타 학습 지원**: 톱10 덱 공개, 리플레이 다운로드

### 3.7 MG-0007: 픽셀 농장 방치RPG

#### 게임 루프
```
농장 관리(수확/심기) → 자원 수집 → 간단한 전투/퀘스트 → 농장/마을 업그레이드 → 반복
```

#### 재미 요소 (Fun Pillars)
| 축 | 목표 | 핵심 경험 |
|----|------|----------|
| 힐링/루틴 (Healing) | 스트레스 없는 플레이 | 하루 피로를 풀어주는 힐링 |
| 성장/확장 (Growth) | 농장 확장 | 작은 밭 → 대농장 성취감 |
| 꾸미기 (Expression) | 개성 있는 농장 | 내 농장에 대한 애착 |

#### 디자인 레버
- **루틴 체크리스트**: 아침(수확→판매→심기), 저녁(동물돌봄→퀘스트→잠자기)
- **루프 연결**: 농장 음식 → 모험 버프, 모험 보상 → 희귀 씨앗
- **NPC 생활감**: 아침엔 빵집 NPC 배달, 비오면 낚시꾼 등장
- **꾸미기 기능성**: 허수아비 → 작물 성장 +3%

```

#### 재미 요소 (Fun Pillars)
| 축 | 목표 | 핵심 경험 |

스테이지 선택 → 퍼즐 플레이 → 클리어 보상 → 진행도 해금 → 반복
```

**핵심 설계 원칙**:
- 클리어율 90% 이상 유지
- 하트/에너지 시스템으로 세션 조절
- 별점(1~3성) 시스템으로 재도전 동기

---

### 6.3 로그라이크형 루프

**적용 게임**: 0005, 0008, 0014, 0020

```
런 시작(초기 셋업) → 진행(선택/보상) → 런 종료(성공/실패) → 메타 성장 → 재도전
```

**핵심 설계 원칙**:
- 런 당 15~25분 설계
- 실패해도 의미 있는 메타 보상
- 빌드 다양성 확보 (상위 5빌드 50% 이하)

### 6.4 PVP/경쟁형 루프

**적용 게임**: 0006, 0013

```
덱/팀 편성 → 매치 참여 → 결과/랭킹 반영 → 시즌 보상 → 반복
```

**핵심 설계 원칙**:
- 디비전/리그 시스템으로 동기 부여
- 메타 순환 설계 (시즌별 밸런스 패치)
- 패배 스트레스 관리 (위로 보상, 매칭 보정)

### 6.5 수집형 JRPG 루프

**적용 게임**: Level A 전체 (0025-0036)

```
전투 → 재화/파편 획득 → 캐릭터 수집 → 성장 → 콘텐츠 진행 → 일일 보상/이벤트 → 반복
```

**핵심 설계 원칙**:
- 가챠/뽑기 시스템 (천장 보장)
- 캐릭터 성장축 다양화 (레벨/승급/장비/스킬)
- 호감도 시스템으로 캐릭터 애착
- 라이브 이벤트로 지속적 신선함

## 9. 레벨 디자인 철학

### 9.1 난이도 곡선 설계

#### 일반 스테이지
```
튜토리얼(1-10) → 성장기(11-50) → 중반(51-100) → 후반(101+)
     ↓              ↓              ↓              ↓
  쉬움(100%)    보통(95%)     어려움(85%)    하드(70%)
```

#### 보스 스테이지
- **패턴 학습**: 첫 도전은 패턴 파악, 재도전으로 클리어
- **전략 요구**: 단순 스탯이 아닌 빌드/전략 필요
- **역할 분담**: 레이드는 역할별 협동 필수

---

### 9.2 콘텐츠 해금 구조

```
메인 스토리 진행 → 스테이지 별점 누적 → 신규 콘텐츠 해금
       ↓
   일일/주간 콘텐츠
       ↓
   이벤트/시즌 콘텐츠
```

---

### 9.3 보상 설계 원칙

| 원칙 | 설명 |
|------|------|
| **즉시 보상** | 클리어 즉시 기본 보상 |
| **누적 보상** | 진행도/마일스톤 보상 |
| **랜덤 보상** | 드롭 확률 기반 (희귀템) |
| **시간 보상** | 방치/복귀 보상 |
