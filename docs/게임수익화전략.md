
# Monthly Games 공통 게임 수익화 전략

## 목표
- 1개월 1게임 출시 구조에 최적화
- 광고 + 인앱결제 + 개인화 기반 LTV 극대화
- 모든 게임에 재사용 가능한 공통 수익화 모듈

## 수익화 4대 축
1. 광고 수익 (Rewarded / Interstitial)
2. 인앱 결제(IAP)
3. 개인화 오퍼
4. 라이브 이벤트 기반 매출

## 광고 전략
- Rewarded Ads 중심
- 개인화된 광고 빈도/보상 계수
- 광고 거부 성향 유저는 보상 강화로 전환 유도

## IAP 전략
- Starter Pack / Value Bundle / Comeback Pack
- 가격대: $0.99 / $4.99 / $9.99
- Pay-to-win 금지, Pay-for-speed 허용

## 개인화 수익화
- 유저 행동 기반 오퍼 자동 선택
- 이탈 위험 유저 → 보상형 이벤트
- 고과금 가능성 유저 → 가치형 번들

## KPI
- ARPDAU
- Ads ARPDAU
- Conversion Rate
- LTV / CPI

## 기술 연동
- Firebase Remote Config
- Firestore UserProfile
- BigQuery 분석
- GA4 이벤트

## 운영 원칙
- 투명성 유지
- 공정성 훼손 금지
- 장기 잔존율 우선
