# Monthly Games — Level A 수집형 JRPG 라인업(12개월) + 재미요소 + 수익화 전략 v1.0
- 작성일: 2025-12-12
- 기준: **Level A(스토어 가능 ‘성인 취향’)** = 노골적/외설적 성적 콘텐츠가 아니라 **성숙한 톤, 관계성, 코스튬 미학, 연출**로 매력 강화
- 플랫폼/스택: Flutter(Flame) · Firebase/GCP · GA4/BigQuery · Spine2D · ComfyUI
- 운영 목표: **월 1개 게임 출시**(템플릿 재사용 + 테마 교체), 출시 후 30일 내 데이터 기반 최적화

---

## 0) 핵심 결론(요약)
- Monthly Games의 정체성은 “게임 1개”가 아니라 **월 1개를 반복 출시/학습하는 시스템**이다.
- 따라서 수집형 JRPG 라인업은 **공통 템플릿 고정** + **테마/캐릭터/연출만 교체**로 설계한다.
- 재미는 “전투”가 아니라 **캐릭터 수집·성장·관계·연출(호감도/코스튬/스토리)**에서 나온다.
- 수익화는 **P2W 금지**를 전제로, **가챠(천장 고정) + 코스튬 + 배틀패스(28일) + 가치형 번들 + 개인화**를 표준화한다.

## 1) Level A 정의 및 가드레일(필수)
### 1.1 Level A(스토어 가능) 설계 원칙
- 성인 취향 = “노골적 성행위/성적 만족 제공”이 아니라 **성숙한 분위기 + 관계성 + 미학적 연출**
- 핵심은 **‘기대감’과 ‘캐릭터 서사’**로 매력을 만들고, 수위는 “암시” 수준을 넘지 않는다.

### 1.2 표현 가드레일(운영/리스크 관리)
- (금지) 미성년으로 해석될 여지가 있는 외형/대사/설정
- (금지) 명시적 성행위 묘사, 성적 만족을 주요 목적으로 하는 표현
- (권장) 성숙한 관계/유머/긴장감/금기 “서사적” 처리
- (권장) 코스튬은 ‘미학/패션/캐릭터성’ 중심(노골적 클로즈업/과도한 강조 피함)
- (필수) 아트/텍스트 생성 파이프라인에 “금지 태그/문구” 자동 필터

## 2) 공통 재미요소(템플릿) — 월 1개 출시를 가능하게 하는 고정 구조
### 2.1 코어 판타지(Core Fantasy)
- “매력적인 전투원들을 수집하고, 성장시키고, 관계를 쌓으며, 위기를 돌파한다.”

### 2.2 재미의 6축(Fun Pillars)
1) **수집 욕구**: 희귀도/클래스/속성/포지션, 픽업/한정/테마 풀
2) **성장 쾌감**: 레벨/승급/스킬/장비/시너지, “막힘→돌파” 설계
3) **전투 연출**: Spine2D 애니메이션, 궁극기 연계(오버드라이브), 타격감(이펙트/사운드)
4) **관계성/호감도**: 개인 스토리, 대사 톤, 선호 트리거(냉소/헌신/장난/금기 등)
5) **메타 루프**: 일일/주간 미션, 파밍 루프, 컬렉션 완성도
6) **라이브 이벤트**: 7일 단위 이벤트(룰 재사용), 28일 배틀패스, 시즌 테마

### 2.3 코어 루프(표준)
```yaml
CoreLoop:
  1_acquire: "전투/이벤트 → 재화/파편 획득 → 캐릭터/장비 수집"
  2_upgrade: "성장(레벨/승급/스킬/장비) → 전투력 상승"
  3_express: "코스튬/편성/궁극기 연출 → 나만의 파티 표현"
  4_progress: "메인 에피소드/보스/레이드 진행"
  5_social: "랭킹/길드(선택) 또는 협력 레이드(선택)"
  6_return: "일일 보상/패스/이벤트로 복귀"
```

### 2.4 “전투 변주 1개” 규칙
- 매달 테마가 바뀌더라도, 전투 시스템은 완전 새로 만들지 않는다.
- 각 게임은 **전투 변주 1개만** 추가한다(예: 부위파괴, 포메이션 시너지, 덱빌딩 로드아웃 등).

## 3) 공통 수익화 전략(템플릿) — 아이덴티티에 맞는 표준 설계
### 3.1 비협상 원칙
- Pay-to-Win 금지(성능 격차 상한)
- 확률/천장/로그 투명성
- 강제 광고 최소화(Rewarded 중심), UX 기반 선택권

### 3.2 수익화 모듈 표준(6종)
1) **가챠(천장 고정)**: 픽업 1종 + 상시 1종(초기 단순화)
2) **코스튬/스킨 샵**: 캐릭터 매력의 핵심 수익원(성능 영향 최소/없음)
3) **배틀패스(28일)**: 월 1개 출시 리듬과 동일, ARPDAU 안정화
4) **번들/오퍼(Starter/Value/Comeback)**: 개인화로 노출/구성 최적화
5) **Rewarded Ads**: F2P 유저의 선택 수익(일일 캡/쿨다운 가드레일)
6) **개인화(Personalization)**: 유저별 “광고/오퍼/난이도/보상계수” 자동 튜닝

### 3.3 가격대(권장)
```yaml
PriceTiersUSD: [0.99, 2.99, 4.99, 9.99, 19.99]
DefaultBundles:
  starter: 0.99
  comeback: 2.99
  value_mid: 4.99
  value_high: 9.99
PassPriceUSD: 9.99
CosmeticsUSD: [1.99, 3.99, 7.99]
```

### 3.4 가드레일(권장)
```yaml
Guardrails:
  pay_to_win: false
  stat_gap_limit_percent: 25
  gacha_pity_required: true
  rewarded_daily_cap: 10
  interstitial_daily_cap: 3
  popup_per_session_cap: 1
  interstitial_first_3min_block: true
```

## 6) 게임별 상세(재미요소 + 수익화 전략 + LiveOps)
> 각 게임은 공통 템플릿을 공유하며, 아래 항목만 테마에 맞게 교체한다.

### M1 — NEON COVERT
- 재미요소
  - 캐릭터 관계: 계약/거래/동맹의 긴장감(대사 톤)
  - 전투 변주: 엄폐/라인 슈팅(2D 사이드), “피격 최소화” 손맛
  - 성장: 무기 모듈(확장 슬롯)로 빌드 차별
- 수익화
  - 코스튬: 네온 패션/작전복/야전 장비(미학 중심)
  - 패스: “작전 시즌” 28일, 유료 트랙에 한정 코스튬 1
  - 가챠: 픽업 1(월간 테마 캐릭터)
- LiveOps
  - 주간 이벤트: “도시 봉쇄전”(룰 재사용), 랭킹은 선택(무압박)

### M2 — WITCH COMPANY
- 재미요소
  - 버프/디버프 체인으로 “콤보 설계” 재미
  - 호감도: 계약 조건/비밀 의뢰로 개인 스토리 강화
- 수익화
  - 픽업 가챠 중심(천장 고정), 코스튬은 “마녀복/의뢰복”
  - 번들: 진행 막힘(재료 부족) 시 Value Bundle 개인화 노출
- LiveOps
  - “의뢰 시즌” 패스, 주간 의뢰(퀘스트 룰 재사용)

### M3 — STEEL SERAPH
- 재미요소
  - 부위파괴/약점 노출: 보스전 연구→성취
  - 레이드 1종(선택): 역할(딜/브레이크/서포트) 명확
- 수익화
  - 레이드 번들(주간): 레이드 재료/장비 강화권(성능 격차 상한)
  - 코스튬: 성당/기계 미학(고급스러운 디자인)
- LiveOps
  - 주간 보스 로테이션(룰 고정, 보상만 교체)

### M4 — SINKING CITY: NOIR
- 재미요소
  - 산소/리스크 자원: 짧은 전투에 “압박감”
  - 선택지 스토리: 신뢰/배신 분기(수집형 관계성 강화)
- 수익화
  - 코스튬(누아르 수트/드레스), 가치번들(진행 정체 타겟)
  - 광고는 최소(Rewarded 2~3 placement)
- LiveOps
  - “침수 지역 수색” 주간 이벤트(재사용)

### M5 — BLOOD OPERA
- 재미요소
  - 턴제(행동게이지): 전술·순서 설계 + 화려한 궁극기
  - 수집: 귀족 세력별 파티 시너지
- 수익화
  - 한정 코스튬 드랍(월 2개), 패스 중심
  - 가챠는 픽업 단순 유지(과도한 배너 난립 금지)
- LiveOps
  - “오페라 시즌” 28일 패스 + 주간 무도회 이벤트

### M6 — DESERT MIRAGE
- 재미요소
  - 포메이션 시너지: 자리(전열/후열)로 역할 차별
  - 방치 요소를 약하게 포함(오프라인 보상) → 월간 유지
- 수익화
  - 성장부스트(시간 단축) 판매 허용(단, P2W 제한)
  - 번들(재료/스태미나) 개인화
- LiveOps
  - “카라반 호위” 주간 이벤트

### M7 — NIGHT SHIFT HOSPITAL
- 재미요소
  - 상태이상 역전: 디버프를 “치유”로 바꾸는 전투 퍼즐
  - 스토리: 미스터리 에피소드(짧게 강하게)
- 수익화
  - 픽업 + 코스튬(유니폼/야근룩 등 ‘직업 미학’)
  - 패스는 “야간 근무 시즌”
- LiveOps
  - “사건 기록” 이벤트(룰 재사용)

### M8 — ARCANE PRISON
- 재미요소
  - 덱형 로드아웃: 스킬 조합으로 빌드 다양성
  - 수집: “룬/카드” 수집으로 성취감(캐릭터+덱)
- 수익화
  - 덱 확장 패스(편의/꾸미기 중심), 코스튬
  - 가챠는 캐릭터 중심 유지(덱으로 과금 폭증시키지 않음)
- LiveOps
  - “감옥 폭동” 시즌 이벤트

### M9 — SKY PIRATE SYNDICATE
- 재미요소
  - 협력 레이드(선택): 역할 분담, ‘타격감’ 강화
  - 파티 시너지(선장/기관/포격/의무)
- 수익화
  - 레이드 패스 + 코스튬(해적단/정장/항공복)
  - 번들은 레이드 재료 중심(성능 격차 상한 유지)
- LiveOps
  - 주간 레이드 로테이션

### M10 — GLASS GARDEN
- 재미요소
  - 타이밍 QTE(세미): 자동 전투를 “손맛으로 보정”
  - 감정/관계 서사: 짧은 개인 스토리 강점
- 수익화
  - 코스튬 중심 + (선택) Premium “광고 제거”
  - 가챠는 단순(픽업 1) 유지
- LiveOps
  - “정원 축제” 시즌 패스

### M11 — UNDERWORLD AUCTION
- 재미요소
  - 리스크-리턴 선택 전투: 보상 높일수록 위험 증가
  - 경제/재화 설계가 재미(획득/소모 루프)
- 수익화
  - 가치 번들(희귀 재료/교환권) 개인화
  - 코스튬은 “정장/경매룩” 고급 미학
- LiveOps
  - “금지 물품 주간” 이벤트

### M12 — RED MOON PROTOCOL
- 재미요소
  - 궁극기 연계(OD): 폭발감, 팀 조합 연구
  - 시즌 피날레 서사: 몰입 강화
- 수익화
  - 시즌 패스 + 한정 픽업(연말 테마) + 한정 코스튬
- LiveOps
  - 연말/시즌 이벤트(보상 인플레 가드레일 필수)
