# Monthly Games 비전

## 미션
한 달에 하나씩, 12개월 동안 12개의 모바일 게임을 출시한다.

## 핵심 가치
1. **속도** - 빠른 프로토타이핑과 출시
2. **학습** - 각 게임에서 배운 것을 다음 게임에 적용
3. **재사용** - 공통 모듈을 통한 효율적 개발

## Year 1 KPI
- 12개 게임 출시
- 총 다운로드 100만 달성
- 공통 모듈 재사용률 70% 이상

## 기술 스택
- **클라이언트**: Flutter + Flame
- **백엔드**: Firebase + Cloud Run
- **분석**: GA4 + BigQuery
- **인프라**: GCP + Terraform
